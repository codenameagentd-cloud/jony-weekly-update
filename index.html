CMC Bug Reporting Skill

v1.2.0 — 5× Faster Filing

Before · ~5 min / bug
  • 到 Linear 手動選擇 Team → Project → Label → Priority
  • 每次填入重複模板欄位（repro steps、版本號、截圖⋯）
  • 中文內容需自己翻譯成英文

  • 中文直接輸入，自動翻譯為英文填入 Linear
  • 截圖貼入路徑或 URL，自動判斷類型
  • 版本號記憶，下次自動帶入
